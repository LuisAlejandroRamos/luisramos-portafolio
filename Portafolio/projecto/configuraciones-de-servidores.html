Configuración de Servidores

Implementación de servicios de red en servidores físicos utilizando Debian.

Debian DNS DHCP Apache2 MariaDB SQL Server Redes LAN

Este proyecto consiste en la configuración completa de una infraestructura de servidores físicos utilizando Debian Linux. Se implementaron servicios esenciales como DNS, servidor web, DHCP, enrutamiento, firewall y bases de datos para dar soporte a sistemas empresariales dentro de una red local.

Servicios configurados

✔ Servidor DNS (BIND9)
Configuración de zonas directas e inversas, dominios internos, reenviadores y resolución local para toda la red.

✔ Servidor Web (Apache2)
Implementación de sitios web internos, paneles de administración, Virtual Hosts y certificados SSL.

✔ Servidor DHCP
Distribución automática de direcciones IP, reservas, gateway, DNS y monitoreo de dispositivos conectados.

✔ Router Físico en Debian
Activación de IP Forwarding, configuración de NAT con iptables, firewall y conexión hacia el proveedor de internet.

✔ Servidor de Base de Datos
Instalación y configuración de MariaDB y SQL Server en entornos físicos, con seguridad, usuarios y acceso remoto para aplicaciones empresariales.

✔ Seguridad
Configuración de firewall, protección de puertos, control de accesos y mantenimiento del servidor.

Resultados del proyecto

La red quedó completamente funcional y segura, con servicios administrativos internos, acceso a dominios locales, distribución automática de IP y servidores estables.
